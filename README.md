# Project 2

**Submission Instructions:** Stage, commit, and push your changed files. Then complete the submission form.

## Client

```sh
cd client
```

**Install dependencies:**

```sh
npm install
```

**Start development server:**

```sh
npm start
```

## Server

```sh
cd server
```

**Install dependencies:**

```sh
npm install
```

**Initialize database:**

```sh
mongosh init.mongo.js
```

**Start development server:**

```sh
npm run dev
```
